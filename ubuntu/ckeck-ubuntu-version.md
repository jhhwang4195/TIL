# 우분투 버전 확인

* lsb_release 명령을 사용하면 ubuntu 버전을 확인할 수 있습니다.
* lsb_release 명령은 특정 LSB (Linux Standard Base) 및 분배 특정 정보를 제공합니다.

## lsb_release 명령 사용하기

$ lsb_release -r
```
Release:	14.04
```

$ lsb_release -a
```
No LSB modules are available.
Distributor ID:	Ubuntu
Description:	Ubuntu 14.04.5 LTS
Release:	14.04
Codename:	trusty
```
